## Installation

Use the Flutter [Pub get]() to install all the dependencies.

```bash
flutter pub get
```

## Instructions

```dart
const api_Key = 'YOUR_API_KEY';//Navigate to Constant.dart and put your API key from (https://www.newsapi.org/).

Hot Restart the app.

```
